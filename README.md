**Splitzy – Effortless Bill Splitting**
Splitzy makes sharing expenses with friends, roommates, or travel groups simple and stress-free. Instantly calculate who owes what, track payments, and keep everyone on the same page. No more awkward math or forgotten debts—Splitzy makes splitting bills fast, fair, and fun.

**Tech Stack for Splitzy**

**Frontend:**

- **React** – For building a dynamic, responsive user interface.
- **Tailwind CSS** – For fast, modern, and customizable styling.

**Backend:**

- **Django REST Framework** – For creating robust APIs to handle user authentication, bill management, and payment tracking.

**Optional / Future Enhancements:**

- **PostgreSQL** – For reliable data storage.
- **JWT Authentication** – For secure user login and authorization.
- **Docker** – For easy deployment and environment management.
